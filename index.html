Introduction to HTML

HTML stands for Hyper Text Markup Language. It is the language that is used to create and structure content on the web by telling the browser how to display it.

The basic structure of a HTML element consists of a start tag, content, and then an ending tag.

  • <tagname>Place content here.</tagname>
Setting Up An HTML Document

The structure of all HTML documents must follow the below guidelines:

  • The first line should be a document type declaration, <!DOCTYPE html>. This tells the browser to expect HTML. The declaration only appears once and must be before the <html> tags.
  • After the declaration, the actual document itself should begin with <html> and end with </html>.
  • Inside the html tags, the <head> tag is added to contain the meta information about the document. This includes the title of the page, as well as links to stylesheets. This tag starts with <head> and ends with </head>.
  • After the <head> tags, and still within the <html> tag, comes the <body> tags. This will contain the visible content of the page and should start with <body> and end with </body>.
